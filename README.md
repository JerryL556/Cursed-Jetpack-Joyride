# Cursed-Jetpack-Joyride

Live game: https://jerryl556.github.io/Cursed-Jetpack-Joyride/

A Phaser-based Jetpack Joyride-style prototype with pixel-art visuals, endless hazards, pickups, and a few cursed twists.

## Features

- Hold `Space` or left mouse to fly upward and release to fall.
- Endless zappers, missiles, and coin lines.
- Second-chance damage system with temporary recovery.
- Secret invincibility toggle on `I`.
- Pickup effects:
- `The Floor Is Lava`
- `Rocket Salvo`
- `Wrong Magnet`

## Controls

- `Space`: fly
- `Left Mouse`: fly
- `I`: toggle secret invincibility mode
- `Space` or `Left Mouse` on death screen: restart

## Pickups

### The Floor Is Lava

Turns the floor into lava for a short time. Landing on it will damage or kill the player.

### Rocket Salvo

Makes missiles spawn much faster for a short time.

### Wrong Magnet

Repels coins away from the player and prevents coin collection while active.

## Tech

- Phaser
- Arcade Physics
- In-code pixel-style textures and UI
